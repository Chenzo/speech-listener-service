Speech Listener Service

Local Windows speech listener for a selected microphone.

What It Does

The service captures microphone audio with ffmpeg DirectShow, sends it to a local Vosk speech recognizer, and broadcasts transcript and trigger events over a local WebSocket server.

Download Release

Download the portable Windows app from this repository's GitHub Releases page.

Current release requirement:

The Electron app bundles the launcher, Node service, and Python/Vosk transcriber. A local Vosk model is still required.

Run From Source

Install Node dependencies:

npm install

Install the Python Vosk package:

py -3 -m pip install vosk

Install the default Vosk model:

npm run install-model

Start the command-line listener:

npm run start

Start the Electron launcher:

npm run start:electron

WebSocket Output

The service listens locally at:

ws://localhost:3011

Example transcript message:

{ "type": "transcript", "text": "hello there", "final": false }

Example trigger message:

{ "type": "trigger", "phrase": "rock", "audio": "rl_short" }

Build Portable App

Create the portable Electron executable:

npm run dist

The output is written to:

dist/Speech Listener-<version>-portable.exe

The build machine needs PyInstaller and the Python vosk package. Release users do not.

Windows-only first version. Separate ffmpeg installation is not usually needed.
