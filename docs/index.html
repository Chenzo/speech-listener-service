Speech Listener Service

Local Windows speech listener for a selected microphone.

Download

  • Download the latest Windows app from GitHub Releases.
  • Download a Vosk speech model from the Vosk models page. We recommend vosk-model-small-en-us-0.15.
  • Extract the Vosk model zip file to a folder you can find again, such as Documents\vosk-model-small-en-us-0.15.

Setup

  • Launch the Speech Listener app.
  • Choose the microphone or audio source you want to listen to.
  • Click the Vosk model chooser and select the extracted model folder, not the original zip file.
  • Adjust the WebSocket port or keyword triggers if needed.
  • Click Start.

The app starts a local WebSocket server at ws://localhost:3011 by default.

Windows-only first version. Separate ffmpeg installation is not usually needed.
